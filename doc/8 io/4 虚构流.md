# 虚构流

## 构建Reader、writer

构建Reader

```go
reader := strings.NewReader("haha")
```

构建Writer

```
writer := bytes.Buffer{}
```

示例：

```go
func TestBuildFakeReader(t *testing.T) {
	writer := bytes.Buffer{}
	reader := strings.NewReader("haha")
	var buf [1024]byte
	for {
		n, err := reader.Read(buf[:])
		if err == io.EOF {
			break
		}
		if err != nil {
			t.Error("read err")
		}
		writer.Write(buf[:n])
	}
	if writer.String() != "haha" {
		t.Error(fmt.Sprintf("except str is haha, but is %s", writer.String()))
	}
}
```
