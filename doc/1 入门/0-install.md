# 安装

## Linux

**方式一**

```shell
yum install -y golang
```

**方式二**

直接下载goxxx.tar.gz，将包解压，将`bin/go`可执行文件放到/usr/bin目录下即可

## windows

方式一：解压tar包

1、解压tar包到/\<go_work_dir>/go

2、添加环境变量path /\<go_work_dir>/go/bin
