Go语言主要有四种类型的声明语句：var、const、type和func，分别对应变量、常量、类型和函数实体对象的声明。

# 变量的声明

```golang
s := ""            // commonly used in function
var s string       // commonly used in overall
var s = ""
var s string = ""

var a int
var a *int
a := int8(1)
```

# 常量的声明

```golang
const boilingF = 212.0
```
